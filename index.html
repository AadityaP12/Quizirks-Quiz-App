About Quizirks

Quizirks is a quiz app that was created as part of my journey into web development, focusing on building interactive and responsive user experiences using core HTML, CSS, and JavaScript. It’s designed not just for fun but also as a learning tool.

As a web development learner, I wanted to create something interactive and visually engaging while reinforcing my understanding of core frontend technologies. This helped me practice DOM manipulation, event handling, and time-based functions in JavaScript and sets the stage for future upgrades using frameworks like React.
